# Siriux User Guide

## Welcome to Siriux

Siriux is a powerful, modern web application platform designed to help you build and manage your digital projects with ease. Whether you're a team leader, project manager, or team member, Siriux provides the tools you need to collaborate effectively.

## Getting Started

### System Requirements

- **Modern Web Browser**: Chrome 90+, Firefox 88+, Safari 14+, or Edge 90+
- **Internet Connection**: Stable broadband connection recommended
- **Device**: Desktop, tablet, or mobile device

### First Time Setup

1. **Open your web browser** and navigate to the Siriux application URL
2. **Create your account** by clicking "Sign Up"
3. **Fill in your information**:
   - First and last name
   - Email address
   - Password (minimum 8 characters)
   - Department (optional)
4. **Check your email** for verification (if enabled)
5. **Log in** with your credentials

## Dashboard Overview

After logging in, you'll see your personalized dashboard:

### Navigation Menu
- **Dashboard**: Your main workspace overview
- **Projects**: View and manage your projects
- **Team**: See team members and their roles
- **Settings**: Configure your preferences
- **Help**: Access documentation and support

### Quick Actions
- **Create New Project**: Start a new project quickly
- **Invite Team Members**: Add people to your team
- **View Recent Activity**: See latest updates across projects

## User Profile Management

### Editing Your Profile

1. **Click on your profile picture** in the top right corner
2. **Select "Profile"** from the dropdown menu
3. **Update your information**:
   - Name and contact details
   - Profile picture
   - Department and role
   - Notification preferences
4. **Click "Save Changes"** to update

### Password Management

1. **Go to Profile → Security**
2. **Click "Change Password"**
3. **Enter your current password**
4. **Enter and confirm your new password**
5. **Click "Update Password"**

## Project Management

### Creating a New Project

1. **Click "Create Project"** from the dashboard or Projects page
2. **Fill in project details**:
   - Project name
   - Description
   - Start and end dates (optional)
   - Project type
3. **Set privacy level**:
   - **Public**: All team members can view
   - **Private**: Only invited members can view
4. **Add team members** (optional)
5. **Click "Create Project"**

### Project Dashboard

Each project has its own dashboard with:

#### Overview Tab
- **Project statistics**: Completion percentage, milestones, team activity
- **Recent updates**: Latest changes and comments
- **Upcoming deadlines**: Tasks due soon

#### Tasks Tab
- **Task list**: All project tasks with status
- **Create new task**: Add assignments, due dates, priorities
- **Filter and sort**: By status, assignee, due date

#### Team Tab
- **Team members**: View all project participants
- **Roles and permissions**: See what each member can do
- **Activity feed**: Recent team actions

#### Files Tab
- **Document library**: All project files in one place
- **Upload files**: Drag and drop or click to upload
- **File sharing**: Share with team members or external partners

### Task Management

#### Creating Tasks
1. **Go to the Tasks tab** in your project
2. **Click "Add Task"**
3. **Fill in task details**:
   - Task title and description
   - Assign to team member
   - Due date and priority
   - Tags or labels
4. **Click "Create Task"**

#### Task Status
- **To Do**: Not started yet
- **In Progress**: Currently being worked on
- **Review**: Ready for review
- **Done**: Completed

#### Task Collaboration
- **Comments**: Add notes and updates
- **Attachments**: Share relevant files
- **Checklists**: Break down tasks into steps
- **Time tracking**: Log hours spent

## Team Collaboration

### Inviting Team Members

1. **Go to Settings → Team**
2. **Click "Invite Members"**
3. **Enter email addresses** of people to invite
4. **Assign roles**:
   - **Admin**: Full access to all features
   - **Manager**: Can manage projects and team
   - **Member**: Can participate in assigned projects
5. **Add personal message** (optional)
6. **Click "Send Invitations"**

### Team Communication

#### @Mentions
- **@username**: Notify specific team members
- **@team**: Notify all project team members
- **@here**: Notify active team members

#### Comments and Discussions
- **Task comments**: Discuss specific tasks
- **Project discussions**: General project conversations
- **File comments**: Discuss uploaded documents

### Activity Tracking

#### Recent Activity Feed
- **Task updates**: Status changes, new assignments
- **File uploads**: New documents added
- **Comments**: New discussions and replies
- **Project changes**: Settings and member updates

#### Notifications
- **Email notifications**: Get updates via email
- **In-app notifications**: See updates when logged in
- **Mobile push notifications**: Get alerts on your phone

## File Management

### Uploading Files

1. **Go to the Files tab** in your project
2. **Click "Upload Files"** or drag and drop files
3. **Choose files** from your computer
4. **Add descriptions** and tags (optional)
5. **Set access permissions**:
   - **All team members**: Anyone can view
   - **Specific members**: Only selected people
6. **Click "Upload"**

### File Types Supported
- **Documents**: PDF, Word, Excel, PowerPoint
- **Images**: JPG, PNG, GIF, SVG
- **Videos**: MP4, AVI, MOV
- **Archives**: ZIP, RAR, 7Z
- **Code files**: TXT, MD, JSON, XML

### File Organization
- **Folders**: Create nested folder structures
- **Tags**: Add descriptive tags for easy searching
- **Versions**: Keep multiple versions of documents
- **Favorites**: Mark important files for quick access

## Search and Filtering

### Global Search

1. **Click the search bar** at the top of any page
2. **Type your search terms**
3. **Use filters** to narrow results:
   - **Content type**: Tasks, files, projects, people
   - **Date range**: Recent or custom date range
   - **Author/Creator**: Filter by specific person
4. **Press Enter** or click the search button

### Advanced Search Operators
- **"exact phrase"**: Search for exact matches
- **-exclude**: Exclude terms from results
- **filetype:pdf**: Search specific file types
- **created:today**: Search by creation date

### Project Filtering

In any project, you can filter:

#### Tasks
- **Status**: To Do, In Progress, Review, Done
- **Assignee**: Specific team members
- **Priority**: High, Medium, Low
- **Due date**: Overdue, today, this week, custom range

#### Files
- **File type**: Documents, images, videos
- **Upload date**: Recent or custom range
- **Uploader**: Specific team member
- **Tags**: Descriptive labels

2. **Choose theme**:

#### Global Shortcuts
- **Ctrl/Cmd + K**: Quick search

## Troubleshooting

### Common Issues

#### Login Problems
- **Forgot password**: Use "Forgot Password" link
- **Account locked**: Contact support
- **Browser issues**: Clear cache and cookies

#### Performance Issues
- **Slow loading**: Check internet connection
- **File upload problems**: Check file size limits
- **Sync issues**: Refresh the page or restart app

#### Notification Issues
- **Not receiving emails**: Check spam folder
- **Push notifications not working**: Enable in device settings
- **Too many notifications**: Adjust notification preferences

### Browser Compatibility

If you experience issues, try:
1. **Update your browser** to the latest version
2. **Clear browser cache** and cookies
3. **Disable browser extensions** temporarily
4. **Try a different browser** to isolate the issue

### Reporting Bugs

When reporting issues, include:
- **Description**: What happened and what you expected
- **Steps to reproduce**: Exact actions to recreate the issue
- **Browser and device**: Your technical environment
- **Screenshots**: Visual evidence of the problem
- **Error messages**: Any error text you see

## Tips and Best Practices

### Productivity Tips

#### Daily Workflow
1. **Start with dashboard**: Review priorities and deadlines
2. **Check notifications**: Address urgent items first
3. **Focus on top priorities**: Use the "My Tasks" view
4. **Update progress**: Mark completed items
5. **Plan tomorrow**: Review upcoming tasks

#### Project Organization
- **Use consistent naming**: Clear, descriptive project names
- **Set realistic deadlines**: Allow buffer time for unexpected delays
- **Regular updates**: Keep team informed of progress
- **Document decisions**: Record important choices and reasons

#### Team Collaboration
- **Clear communication**: Be specific in comments and updates
- **Regular check-ins**: Schedule consistent team meetings

### Security Best Practices

#### Account Security
- **Strong passwords**: Use unique, complex passwords

#### Data Security

- **Public Wi-Fi**: Avoid using on unsecured networks
